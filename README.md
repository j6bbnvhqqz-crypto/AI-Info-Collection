# 🧠 AI 信息收集系统

一个基于 Obsidian 的人工智能信息收集与知识管理系统。

## 设计理念

AI 领域发展极快，每天都有大量新模型、新工具、新论文、新观点涌现。这个知识库提供一套结构化的信息收集流程：

**收集 → 暂存 → 分类 → 消化 → 关联 → 输出**

## 目录结构

```
├── Inbox/              # 快速收纳暂存区
├── Daily/              # 每日 AI 快讯
├── 1_AI动态/           # 行业新闻、产品发布、融资
├── 2_模型与论文/        # 新模型、重要论文、技术突破
├── 3_工具与产品/        # AI 工具评测、产品体验
├── 4_公司与人/          # 公司动态、关键人物
├── 5_观点与评论/        # 深度分析、趋势评论
├── Templates/          # Obsidian 模板
├── assets/             # 图片与附件
├── docs/               # 文档
└── outputs/            # 输出物
```

## 使用方式

1. 用 Obsidian 打开本仓库文件夹作为 Vault
2. 看到感兴趣的内容 → 快速丢进 `Inbox/`
3. 每天/每周清理 Inbox → 归入对应分类文件夹
4. 使用 `[[双向链接]]` 和 `#标签` 建立知识网络
5. 定期回顾 `00_AI信息收集MOC` 地图页

## 常用标签

| 标签 | 含义 |
|------|------|
| `#待处理` | 需要进一步消化的内容 |
| `#重要` | 高价值信息 |
| `#趋势` | 行业长期趋势信号 |
| `#工具` | 实用工具和产品 |
| `#模型` | 模型与技术 |
| `#日刊` | 每日快讯 |
